# BotKuisEko
Bot Project
🎮 1️⃣ Bot Ekonomi Mini (Seru Banget)
💡 Konsep:
Setiap user punya uang virtual.
Fitur
!balance → cek uang
!kerja → user dikasi quiz, adda beberapa tingkat kesulitan, kalo bener user dikasi uang, kalo salah ga dapeet. useer bisa pilih tingkatan
!transfer @user jumlah
!Leaderboard terkaya → nunjukin peringkat uang terbanyak


library yg diperluin
1. disscord.py
2. random
3. asyncio/datetime
4. json

struktur folder
1. logic.py → isi logika keuangan dan quiz yg ditampilkan
2. bot.py → isi kode bot
3. quiz.json → bank soal soal
4. users.json → data uang user (kalo user lebih dari 1)
